# Nixと他のツールとの組み合わせに関するレビュー論文の草案

## 書く内容

### 概要

複数のプロジェクトおよび開発チームのパターンに対して
Nixと他のツールを複数の割合で混ぜた開発環境を用意し、それぞれの評価をまとめる。

### 評価指標

- 再現性
- 利便性 (開発体験の良さ)
- 柔軟性

### パターン

#### プロジェクト (仮)

- R言語のパッケージの開発プロジェクト
- pythonのパッケージの開発プロジェクト

### 開発チーム (仮)

- Nix経験者の割合いでパターン分け

### 他のツール (仮)

- Docker
- renv
- uv
